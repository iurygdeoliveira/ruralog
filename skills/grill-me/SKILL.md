---
name: grill-me
description: Entreviste o usuário implacavelmente sobre um plano ou design até atingir um entendimento compartilhado, resolvendo cada ramo da árvore de decisão. Use quando o usuário quiser testar o estresse de um plano, ser questionado sobre seu design ou mencionar "grill me".
---

Entreviste-me implacavelmente sobre cada aspecto deste plano até atingirmos um entendimento compartilhado. Percorra cada ramo da árvore de design, resolvendo as dependências entre as decisões uma a uma. Para cada pergunta, forneça sua resposta recomendada.

Faça as perguntas uma de cada vez.

Se uma pergunta puder ser respondida explorando a base de código, explore a base de código em vez de perguntar.
